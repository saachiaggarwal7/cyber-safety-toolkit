# Cyber Safety Toolkit

A collection of beginner-friendly Python utilities focused on cybersecurity, Linux administration, and security automation.

These tools were built while learning Python, Linux, and cybersecurity concepts. Each project solves a practical security-related task and demonstrates core programming skills.

---

## Tools Included

### Password Strength Checker
Evaluates password strength based on:
- Minimum length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

---

### File Permission Checker
Checks Linux file permissions and reports:
- File or directory type
- Read, write, and execute permissions
- Octal permissions
- Symbolic permissions
- Warning for insecure world-writable files

---

### File Backup Tool
Creates timestamped backups while:
- Preserving file metadata
- Creating backup folders automatically
- Logging backup operations

---

### Log Analyzer
Analyzes log files by:
- Counting ERROR, WARNING, and INFO entries
- Searching keywords
- Detecting failed login attempts
- Counting login attempts by IP
- Generating security reports

---

### User Creation Automation
Automates Linux user administration:
- Checks existing users
- Creates new users
- Sets passwords securely
- Adds users to Linux groups
- Handles common input errors
